## Description

This map displays potential market areas with interactive hover information showing:
- **Market Shape Area**: Geographic extent (percentile)
- **Market Kcal Density**: Agricultural productivity per unit area (percentile)
- **Market Harbors**: Number of accessible harbors (value and percentile)
- **Market Population (1800)**: Estimated population (value and percentile)

### Features
- Hover over any region to see detailed statistics
- Color-coded by Kcal Density percentile (purple = low, yellow = high)
- Responsive design works on desktop and mobile
- Based on historical geographic and economic data
